## 程序更容易理解

函数或者方法输入输出的参数类型，外部条件？

- 使用`js`需要手动调试（console.log....)
- 使用`ts`代码本身就可以回答上诉问题

## 更高的效率

- 在不同的代码块和定义中进行跳转
- 代码自动补全
- 丰富的接口提示

## 更少的错误

- 编译期间能够发现大部分错误
- 杜绝一些常见的错误

## 非常好的包容性

- 完全兼容`js`
- 第三方库可以单独编写类型文件
- 大多数项目都支持`ts`

## 一些小缺点

- 学习成本
- 开发成本
